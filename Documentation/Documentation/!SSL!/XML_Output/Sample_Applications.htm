Sample Applications

There are currently two sample applications provided with Reactor which are located in the ReactorSamples directory.  These applications have a ReadMe.txt file in their root directory.  Be sure to read these and follow the instructions to configure and test the application.

Contact Manager

The Contact Manager sample uses Reactor to create simple contact manager application.  This demonstrates a simple usage of Reactor.

Blog

The Blog sample application uses Reactor and the Model-Glue framework to create a more complex and fully-functional blog application.

 

For the Blog application to work you will have to download the Model-Glue framework from http://www.model-glue.com and create a mapping named “/ModelGlue” which points to the core Model-Glue files.
