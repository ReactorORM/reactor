Overview

This section is intended to show you some of the features of Reactor while also showing a simple example of how Reactor works.  This intended to be followed along with, without requiring you to know much about Reactor.

 

Once you've seen reactor working, and written a bit of code, the rest of the documentation should make a lot more sense.

 

This example doesn't really do anything other than illustrate concepts.  However, the objects created roughly relate to concepts you might find in a typical application.

 

If you want to see code that does something more useful see the Blog and ContactManager sample applications.
